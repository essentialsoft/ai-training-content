# AWS Solutions Architect Associate Certification Prep

**Difficulty** : Intermediate
**Topics** : AWS, Fundamentals, Workflow

## Module Overview

This module prepares you for the AWS Certified Solutions Architect – Associate (SAA‑C03) certification. You’ll build understanding of how to design, deploy and manage scalable, secure AWS architectures that align with ESI’s AI-powered software development cycle. The selected courses combine theory, labs, and exam-prep resources — ideal for solutions-architects, cloud engineers and technical leads.

## Learning Objectives

* Explain the AWS shared responsibility model and the AWS Well-Architected Framework.
* Design and deploy resilient, cost-effective AWS architectures using core services (compute, storage, networking).
* Apply identity, access management and encryption best practices in AWS.
* Troubleshoot and optimize AWS architectures for performance, reliability and cost.
* Prepare for and pass the AWS Certified Solutions Architect – Associate exam.
