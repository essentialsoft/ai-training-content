# AWS Certified AI Practitioner Training Module

**Difficulty** : Beginner

**Topics** : Fundamentals, AI Tools, RAG, Prompt Engineering, Multimodal

## Module Overview

This module guides learners through the core concepts and AWS services needed for the AWS Certified AI Practitioner (AIF-C01) certification. The courses cover foundational AI/ML theory, generative AI with AWS, prompt engineering, and practical hands-on labs using services like Amazon SageMaker and Amazon Bedrock. This knowledge supports ESI’s AI-powered software development cycle — from idea and prototyping to deployment and maintenance.

## Learning Objectives

* Explain key AI, ML, and generative AI concepts and terminology.
* Demonstrate how to use AWS AI/ML services to design and deploy AI solutions.
* Apply prompt engineering and RAG techniques with foundation models on AWS.
* Evaluate security, governance, and compliance considerations in AI workflows on AWS.
* Prepare confidently for the AIF-C01 certification exam using real-world labs and practice tests.
