# AWS Data Engineering Certification Training Module

**Difficulty** : Intermediate

**Topics** : Fundamentals, Workflow

## Module Overview

This module provides a targeted pathway to prepare for the AWS Certified Data Engineer – Associate certification (DEA-C01). Learners will deepen their understanding of key data engineering workflows on AWS—ingestion, storage, transformation, orchestration and governance—within ESI’s AI-powered software development lifecycle. The focus is on practical pipeline design and operational readiness.

## Learning Objectives

* Explain the lifecycle of data from ingestion to serving in AWS environments.
* Build and deploy end-to-end data pipelines using AWS services.
* Optimize AWS data stores and data models for performance, cost and scalability.
* Apply governance, security and compliance practices in AWS data engineering workflows.
* Evaluate readiness for the DEA-C01 exam and align your career pathway accordingly.
