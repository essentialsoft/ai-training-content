# AWS Generative AI Developer Certification Preparation

**Difficulty** : Advanced

**Topics** : AI Frameworks, AI Tools, AI in Software Development, Prompt Engineering, RAG, Workflow

## Module Overview

This module prepares experienced cloud developers and technical leads for the AWS Certified Generative AI Developer - Professional (AIP-C01) exam. You'll combine a deep, hands-on Udemy course with an AWS Skill Builder domain review to design, build, and operate production-grade GenAI solutions on AWS that align with ESI's AI-powered software development lifecycle.

## Learning Objectives

* Apply AWS generative AI services such as Amazon Bedrock and Amazon SageMaker to build and deploy production-ready applications.
* Design and optimize RAG workflows using embeddings, vector stores, and knowledge bases to support software delivery at ESI.
* Employ advanced prompt engineering and workflow orchestration patterns to improve reliability and maintainability of GenAI features.
* Evaluate and troubleshoot GenAI systems for cost, performance, safety, and governance across the full application lifecycle.
* Map exam domains and objectives to concrete architectures, implementation patterns, and operational practices on AWS.
