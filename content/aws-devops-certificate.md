# AWS DevOps Engineer Professional Certification Prep

**Difficulty** : Advanced

**Topics** : Workflow, Testing, Documentation

## Module Overview

Prepare for the AWS Certified DevOps Engineer – Professional (DOP-C02) exam. This module focuses on CI/CD automation, infrastructure as code, monitoring, incident response, and resilient architectures on AWS. It supports ESI’s AI-powered development cycle by strengthening cloud workflows used by platform engineers, SREs, and DevOps teams.

## Learning Objectives

* Identify the six exam domains and map them to core AWS services.
* Build automated CI/CD pipelines using AWS CodePipeline, CodeBuild, and CodeDeploy.
* Apply infrastructure as code with CloudFormation and related tooling.
* Evaluate monitoring, logging, and incident response strategies on AWS.
* Optimize resilient, secure architectures aligned to ESI delivery workflows.
